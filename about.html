About CampusStay Hub

A student project by Victoria University — Professional Certificate in Software Engineering

Background

Victoria University, located in Kampala, Uganda, attracts a large and growing student population each academic year. As enrolment increases, the need for affordable and conveniently located accommodation becomes increasingly critical. Many students — particularly those relocating from outside Kampala — faced considerable difficulty identifying suitable hotels, motels, or hostels within close proximity to campus.

The process of finding accommodation was largely manual, relying on physical visits, informal social networks, and printed notices. No centralised digital platform existed to aggregate and present accommodation options in a structured, easily accessible manner.

CampusStay Hub was conceived to address this gap — providing a single web-based platform where students can search and compare nearby accommodation across multiple types and price ranges, instantly and for free.

What the Platform Does

🔍

Live Search

Search across all listings by name or location as you type — results update instantly without reloading the page.

🎛

Smart Filters

Filter by accommodation type (hotel, motel, hostel) and price range simultaneously, with real-time results.

📞

Direct Contact

Each listing shows pricing, location, and direct contact details including phone and WhatsApp links.

📱

Mobile Friendly

Fully responsive design that works on smartphones, tablets, and desktop browsers without any issues.

Technologies Used

CampusStay Hub was built using three core front-end web technologies:

HTML5 defines the structure of all pages including the accommodation cards, search bar, and filter controls. CSS3 handles all visual styling including the blue and white colour scheme, responsive card layout, and animations. JavaScript powers all interactive functionality including live search, dynamic filtering, real-time card rendering, and DOM manipulation.

No server-side backend or database management system is used in this version. All accommodation data is structured as JavaScript objects on the client side, enabling fast interactions without server round-trips.

Scope & Limitations

CampusStay Hub covers hotels, motels, and hostels within a reasonable distance from Victoria University's main campus in Kampala. In its current form, the platform is a static web application. Listings cannot be updated by providers without developer involvement, and the platform does not support online booking, payment processing, or real-time availability data.

Future Development

Planned future enhancements include migrating accommodation data to a backend database (MySQL or MongoDB), implementing a provider self-registration portal, adding a student review and rating system, integrating Google Maps for visual location guidance, and developing a companion mobile application. Machine learning-based accommodation recommendations are also envisioned as a long-term feature.
